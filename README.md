# 20230420 登入系統 vector 版（login-vector.cpp）

請使用 vector 撰寫一個系統，可以：
* 註冊
* 登入，帳密錯誤時顯示錯誤
* 登入後修改密碼

可以試著加一些功能，例如：
* 註冊時增加姓名欄位，登入後顯示歡迎詞
* 註冊時檢查帳號是否已存在
